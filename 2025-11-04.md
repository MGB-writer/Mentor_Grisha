GIT. Как подключиться к удалённому репозиторию (GitHub) по SSH?
- Сгенерировать ssh-ключ: `ssh-keygen` → будут созданы два файла в "~/.ssh" с названиями "id_ed25519" и "id_ed25519.pub".
- Открыть настройки GitHub: GitHub → Settings → Access → SSH and GPG keys → SSH keys → New SSH key.
- Добавить ключ: В поле "Title" указать название репозитория и пользователя в формате "<user> (<project_name>)"; В поле "Key" вставить строку из файла "*.pub".
- Склонировать репозиторий `git clone <ssh_url>`.

# Bash

Bash. Какова структура условного выражения?
if <condition>; then
    <action>
[elif <condition>; then
    <action>]
[else
    <action>]
fi

Bash. В чём разница между одинарными "[]" скобками и двойными "[[]]"?
Одинарные скобки - устаревший метод, двойные скобки - современный метод (рекомендуемый).
Функционал одинарных скобок беднее и неудобнее по сравнению с двойными скобками. В случае одинарных скобок переменные, команды и выражения надо заключать в двойные кавычки, в случае двойных скобок не надо.

Bash. Какова самая распространённая ошибка при прописывании условия?
Забыли пробелы между условием и квадратными скобками.
Неправильно: [[<condition>]]
Правильно: [[ <condition> ]]
