Python. Что такое оператор "and"?
Это логический оператор, который объединяет условия и возвращает True, если все условия вернули True.

Python. Что такое таблица истинности оператора "and"?
В ней перечислены выражения, соединённые оператором and, показаны все возможные комбинации истинности и ложности и приведены результирующие значения выражений.
True and True -> True
False and True -> False
True and False -> False
False and False -> False

Python. Что такое оператор "or"?
Это логический оператор, который объединяет условия и возвращает True, если хотя бы одно условие вернуло True.

Python. Что такое таблица истинности оператора "or"?
В ней перечислены выражения, соединённые оператором or, показаны все возможные комбинации истинности и ложности и приведены результирующие значения выражений.
True or True -> True
True or False -> True
False or True -> True
False or False -> False

Python. Что такое оператор "not"?
Это логический оператор, который инвертирует (т.е. заменяет на противоположный) результат логического выражения.
False -> True
True -> False

Python. В чём отличие этих импортов модулей?
import <module> / from <module> import * / from <module> import <method>
- "import <module>" устанавливает соединение с модулем и при использовании методов модуля нужно будет указывать его источник (префикс) через точку "<module>.<method>()".
- "from <module> import *" полностью копирует модуль исключая необходимость указывать источник (префикс) через точку "<method>()".
- "from <module> import <method>" копирует только указанный метод.
