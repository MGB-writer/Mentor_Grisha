Python. Что такое таблица истинности оператора "or"?

Python. В чём отличие этих импортов модулей?
import <module> / from <module> import * / from <module> import <method>
- "import <module>" устанавливает соединение с модулем и при использовании методов модуля нужно будет указывать его источник (префикс) через точку "<module>.<method>()".
- "from <module> import *" полностью копирует модуль исключая необходимость указывать источник (префикс) через точку "<method>()".
- "from <module> import <method>" копирует только указанный метод.

Python. Как удалить определённые символы из строки?

По умолчанию удаляет пробелы, табов "\t" и переносов строк "\n" и т.д.
"---hello---" / "-" / "hello"
".,!?привет!?.," / ".,!?" / "привет"

Python. Как разбить строку на множество подстрок?
'<string>'.split([sep='<separator>'][, maxsplit=<number>]) # Слева направо
'<string>'.rsplit([sep='<separator>'][, maxsplit=<number>]) # Справа налево

sep (разделитель) - по умолчанию в качестве разделителя выступают пробел, таб "\t" и перенос строки "\n".
maxsplit (максимальное количество разбиений) - по умолчанию максимальное количество разбиений равняется количеству символов разбиения.

По умолчанию метод "split" идёт слева направо, но если в maxsplit указать "-1", то будет идти справа налево.

Python. Как разбить текст на абзацы?
